# 速连VPN2 - 快速开始指南

## 🚀 5分钟快速上手

### 第一步：配置订阅地址

1. 打开应用，点击底部导航栏的 **"设置"**
2. 在"订阅设置"区域，点击 **"订阅地址"**
3. 在弹出的对话框中输入你的订阅链接
4. 点击 **"保存"**

> 💡 订阅链接格式通常为：`https://example.com/sub?token=xxxxx`

### 第二步：获取节点列表

1. 返回 **"首页"**
2. 点击右上角的 **刷新图标** ↻
3. 等待节点列表加载完成
4. 看到提示 "已更新 X 个服务器" 表示成功

### 第三步：选择并连接

1. 在首页可以看到当前选中的节点
2. 如需更换，点击 **"服务器"** 标签页
3. 浏览节点列表，点击任意节点进行选择
4. 可选：点击 **"全部测速"** 找到最快的节点
5. 返回首页，点击中央的 **大按钮** 连接

### 第四步：验证连接

连接成功后，你会看到：
- ✅ 按钮变为绿色，显示电源图标
- ✅ 状态卡片显示 "已连接"
- ✅ 流量统计开始更新
- ✅ 通知栏显示VPN运行状态

## 🔧 常见问题

### Q1: 提示"请先在设置中配置订阅地址"？

**原因**：首次使用未配置订阅链接

**解决**：
1. 进入设置页面
2. 点击"订阅地址"
3. 输入有效的订阅链接
4. 保存后返回首页刷新

### Q2: 连接超时或失败？

**可能原因**：
- 网络连接问题
- 节点已失效
- 防火墙阻止

**解决方法**：
1. 检查手机网络是否正常
2. 尝试其他节点
3. 点击"全部测速"找到可用节点
4. 重启应用重试

### Q3: 如何找到最快的节点？

**方法一**：自动选择
1. 进入"服务器"页面
2. 点击顶部 **"全部测速"** 按钮
3. 等待测速完成
4. 列表会自动按延迟排序
5. 第一个就是最快的节点

**方法二**：手动测试
1. 点击任意节点右侧的 **闪电图标** ⚡
2. 查看该节点的延迟
3. 选择延迟最低的节点

### Q4: 流量统计不准确？

**说明**：当前版本使用估算值

**原因**：
- 上行流量：准确统计（从TUN设备读取）
- 下行流量：基于上行估算（系数1.2）

**后续改进**：未来版本将集成v2ray API获取精确数据

### Q5: 后台运行被杀掉？

**解决方法**：
1. 将应用加入白名单
2. 关闭电池优化
3. 锁定最近任务中的应用

**操作步骤**（以MIUI为例）：
- 设置 → 应用管理 → 速连VPN → 省电策略 → 无限制
- 最近任务 → 长按应用 → 锁定

### Q6: 如何切换主题？

1. 进入"设置"页面
2. 找到"外观"区域
3. 点击"主题"下拉菜单
4. 选择：深色 / 浅色 / 跟随系统

## 💡 使用技巧

### 技巧1：快速切换节点
- 在首页直接点击节点卡片
- 会跳转到服务器列表
- 选择后立即生效（需重新连接）

### 技巧2：批量测速省时
- 首次使用时建议全部测速
- 测速结果会保存在本地
- 下次启动自动加载历史数据

### 技巧3：国内直连省流量
- 默认开启"绕过大陆地址"
- 访问国内网站不走代理
- 节省流量且速度更快

### 技巧4：断线自动重连
- 默认开启自动重连功能
- 网络波动时自动恢复
- 无需手动操作

## ⚙️ 高级设置

### 本地端口配置
- **Socks5端口**：10808（默认）
- **HTTP端口**：10809（默认）

> 一般无需修改，除非与其他应用冲突

### DNS服务器
- 默认：8.8.8.8（Google DNS）
- 可改为：1.1.1.1（Cloudflare）或 114.114.114.114（国内）

### UDP转发
- 默认开启
- 部分游戏和视频应用需要
- 如遇到问题可尝试关闭
